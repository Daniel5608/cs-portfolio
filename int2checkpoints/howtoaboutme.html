How I Made My About Me Page

Navbar

The Navbar was there to serve as an easy way to switch from the about me to the "How To" page. I made it by using a list and linked the data inside the list to the 1st and 2nd page so that when you click on whichever one it takes you to the page you want it too, using <ul> and I then made it look nice by going to my css file by taking away the bullet points by using "list-style:none;" in my .navbar li class, a border to make it distinct from just text in the background. I then created a more circular border using "border-radius": 13px; because that way it looks way more appealing to the user. I also made it so that when you hover over the data in the navbar it changes color to show that you are hovering over it. I did all this with the code to the right.

Background Color and Border


For the background color I used an image of lightblue by entering the code of "background-image: url();" because it was the exact color I wanted and it would have been hard to get the exact same color on the rgb scale in numbers. Then I wrapped everything inside the box div which is all the text and the images excluding the navbar, and I created a thick border of 15 pixels of solid light-blue. I made the background inside the border white, using "background-color: white;", because it goes well with the lightblue on the outside of the border. I mostly only used css for these processes.

Image

For the images that I inserted into my about me page I used the url and inserted it into my page with the code of "<img src="the url of the image" width="#" height="#">" The positioning and the height and width you just add width and height and put an equal sign after that and add quotes and add values inside the quotes. To finish, like always close the tag.

Paragraph tags


I used paragraph tags after every subtitle or h2 because it is great for organization and you can add classes to the p tags to affect a certain p tag with an attribute. They also automatically skip a line for you, so you don't have to enter a <br> tag.

Header


I used 3 types of header in my about me page. I used the h1 tag to create the title of the page which read "Who Am I? What have I learned?" to emphasize it is what the whole page is going to be about. The h3 tags I used for subtitles of which I used to tell what I learned and then I used the p-tags to to explain the subtitle. The h3's were also there to show what that paragraph or division would be about. The last header was the h4 I used which was for the title of the are of where I included links for users to learn more about what I had learned. Headers are useful for titles of certain parts of your webpage. They go all the way to h6 for different occurences or parts of the website.

Align: Right, Left, Center and Inline Block

I aligned most of my text on the left of side of the div, box, by entering "text-align:left;" into the cssand everything other than the title is not in the center. I did this using the code "text-align:center;" in the css file. I did not align anything on the right but if you wanted to you would enter "text-align:right;". I added my image at the very top, but at first it was under the text because I had not added "display:in-line block." I later added this and it put the image to the right of the text like it is now. This is because this piece of code tells the webpage to show the two elements on the same line.

Margin and Padding

On my page I used margins to seperate the background of the box and the background of the text and the background of the text inside the box. The margins also display the size of the box and how long, tall, wide, or short it is. To use margins you just enter the code"margin-left:#px or margin: #%;" and "margin-right:#px or margin: #%;" to control the sides of your box and the percent is the percentage of how much screen it takes up, while pixels stay the same for every screen. The padding, how-ever determines the distance of the text and the contents of the box from the edges of the it. It makes your webpage better-looking, and it looks more proffessional when your contents are not touching the walls of your margins and the border. To use padding you just use "padding:#px". Where the hashtag is is the value which determines the distance between the words and the visible border.
